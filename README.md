Just Keep Rolling
By Beau, Kenny, and Clark

Play levels as a rolling ball! 
Try to get the highest score by
not falling, collecting coins,
and/or finishing a level quickly!

TO OPEN GAME

Download the ZIP. Unzip the ZIP.

MAC: Open file "Play Just Keep Rolling MAC"

WINDOWS INTEL: Open file "Play Just Keep Rolling WIN INTEL"

WINDOWS ARM: Open file "Play Just Keep Rolling WIN ARM"



INFO AND CONTROLS:

*Default controls*

Use ASDW or arrow keys to move the 
ball, and use the mouse to pivot 
the camera.

*Alternate controls*

Press M at the start of each level to
control the ball with arrow keys 
and the camera pivot with ASDW 
instead of the mouse.

*Can't finish a level?*

Click L after at least 20 restarts to
skip to the next level.

*Stuck?*

Press R to respawn.
