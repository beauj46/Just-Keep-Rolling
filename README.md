Just Keep Rolling
By Beau, Clark, Kenny

Play levels as a rolling ball! 
Try to get the highest score by
not falling, collecting coins,
and/or finishing a level quickly!

TO OPEN GAME

Download the ZIP (MAC, WIN INTEL, or WIN ARM).

Extract.

MAC: Open file "Just Keep Rolling MAC"

WINDOWS INTEL: Open file "Just Keep Rolling.exe"

WINDOWS ARM: Open file "Just Keep Rolling.exe"



INFO AND CONTROLS:

*Default controls*

Use ASDW or arrow keys to move the 
ball, and use the mouse to pivot 
the camera.

*Alternate controls*

Press M at the start of each level to
control the ball with arrow keys 
and the camera pivot with ASDW 
instead of the mouse.

*Can't finish a level?*

Click L to skip to the next level.

*Stuck?*

Press R to respawn.

*Got a jump power up?*

Click spacebar to use it!
